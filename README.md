**这是一个基于[SkinRestorer](https://github.com/SkinsRestorer/SkinsRestorer/)开发的一个设置玩家进入服务器默认皮肤的插件，非常有效的解决了[BetterHud](https://github.com/toxicity188/BetterHud/)在离线版的情况下玩家可能无法获取头像的问题**

##  版本更新
